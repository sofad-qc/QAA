# Accessibilité et Qualité

Guide d'assurance qualité et d'accessibilité pour les produits développés à la SOFAD.  

- [Liste](./Liste.md) des critères d'acceptation
  - Accessibilité :  [W3C WCAG - AA](https://www.w3.org/TR/2006/WD-WCAG20-20060427/appendixB.html)
  - Assurance qualité :  [OpQuast](https://checklists.opquast.com/en/web-quality-assurance/)

## Buts

- Assurer la qualité constante des produits ;
- Viser l'accessibilité universelle ;
- Offrir une source collaborative et commune et co-créer ;
- Standardiser certaines pratiques d'intégration ;
- Démontrer la réputation de leader de la SOFAD ;
- Réduire les coûts et temps requis par la production ;

## Contenus

- Le dossier [exemples/css](./exemples/css) contient des références CSS pour faciliter l'intégration ;
